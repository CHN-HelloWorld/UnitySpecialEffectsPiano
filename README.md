# Unity 特效钢琴交互系统

一个基于 Unity 引擎 2022.3.23 版本开发的沉浸式钢琴交互系统，整合了粒子特效、动态灯光和物理音效等多媒体元素，提供视听结合的演奏体验。

## ✨ 核心特性

- **琴键交互系统**  
  带有物理反馈的虚拟琴键，支持力度感应与触发动效
- **音符生成系统**  
  实时生成音符粒子特效并触发对应音阶
- **动态灯光系统**  
  根据演奏节奏自动变化的场景灯光效果

## 🛠 技术栈

- Unity 2022.3.23
- C# 脚本编程
- Universal Render Pipeline (URP)
- 粒子系统 (Particle System)
